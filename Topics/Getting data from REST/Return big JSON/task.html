This problem is fully supported by version 2021.9 of the EduTools plugin and newer. To avoid possible errors, please make sure that the plugin is updated. Alternatively, you can solve it using the web interface.

In this task, you need to implement a program that handles GET /colors requests and returns a JSON with information about two colors.

This task can be solved by simply returning one hardcoded String, but try to use custom classes, Map, and lists/arrays. Understanding how this works will help you when creating projects.

Here is the JSON that endpoint GET /colors should return:

{
   "colors":[
      {
         "color":"black",
         "category":"hue",
         "type":"primary",
         "code":{
            "rgba":[
               0,
               0,
               0,
               1
            ],
            "hex":"#000"
         }
      },
      {
         "color":"white",
         "category":"value",
         "type":"primary",
         "code":{
            "rgba":[
               255,
               255,
               255,
               1
            ],
            "hex":"#FFF"
         }
      }
   ]
}

The server should respond with 200 (OK) status code.
